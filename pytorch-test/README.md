Available images:

* [PyTorch 1.9, CUDA 11.1](pytorch1.9-cuda11.1)
* [PyTorch 1.8.1, CUDA 11.1](pytorch1.8.1-cuda11.1)
* [PyTorch 1.6, CUDA 10.1](pytorch1.6-cuda10.1)
